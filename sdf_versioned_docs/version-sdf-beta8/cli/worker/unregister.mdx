# `sdf worker unregister`

The `sdf worker unregister` un-registers a remote worker


### `sdf worker unregister` command

The `worker unregister` command has the following hierarchy:

```bash copy="fl"
$ sdf worker unregister -h
Usage: sdf worker unregister <NAME>

Arguments:
  <NAME>  

Options:
  -h, --help  Print help
```
