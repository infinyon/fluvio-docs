# `sdf worker switch`

The `sdf worker switch` changes the worker in context for other operations.

### `sdf worker switch` command

The `worker switch` command has the following hierarchy:

```bash copy="fl"
$ sdf worker switch -h
Usage: sdf worker switch <NAME>

Arguments:
  <NAME>  

Options:
  -h, --help  Print help
```
