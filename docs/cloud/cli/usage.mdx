Query the CPU and memory usage of your SPUs.

## `fluvio cloud cluster usage`

Example usage:

```bash copy="fl"
$ fluvio cloud cluster usage
Current organization: inf-ops
Using cluster inf-dashboard based on Fluvio Profile

 SPU     CPU    Memory 
 main-0  0.06%  4560Ki 

```
